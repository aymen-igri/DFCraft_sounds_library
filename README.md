# MindCraft Sounds Library

Curated ambient sound library for the MindCraft browser extension.

## 🎧 About

This repository hosts ambient sounds for focus, relaxation, and meditation. All sounds are sourced from Pixabay and are free for commercial use.

## 📂 Categories

- **Rain** - Soothing rain and storm sounds
- **Nature & Forest** - Outdoor ambiences and wildlife
- **White Noise** - Pure focus sounds
- **Café & Indor** - Calming meditation sounds
